Intresseanmälan

HPE Morpheus User Group

Är du intresserad av att gå med i HPE Morpheus User Group? Fyll i din e-postadress så hör vi av oss med mer information om kommande träffar, diskussioner och aktiviteter.

När formuläret skickas sparas intresseanmälan via Formspree så att vi kan följa upp med mer information om HPE Morpheus User Group.
Det här formuläret visar gränssnittet. För att faktiskt spara adresser behöver du koppla det till till exempel Formspark, Netlify Forms, Formspree eller ett eget backend-API.
Genom att skicka in formuläret godkänner användaren att bli kontaktad om HPE Morpheus User Group. Vi använder e-postadressen enbart för att hantera intresseanmälan och återkoppling kring gruppen.
